Privacy Policy for SandCat

Last updated: May 16, 2026

1. Introduction

SandCat is a browser extension designed to assist with aviation data lookup, ADS-B analysis, live aviation overlays, airport and procedure search, workflow utilities, and related aviation research tools.

2. Information SandCat Processes

Local Data Storage

SandCat may store data locally on your device using browser storage. This may include:

  • User preferences and settings
  • Cached aviation datasets, including airports, runways, waypoints, navaids, procedures, frequencies, and visual reporting points
  • Temporary session data, such as selected aircraft, active flight context, saved search state, or extension UI preferences

This data remains on your device unless a feature explicitly requires a user-directed request to an external aviation or utility service.

Clipboard Access

SandCat may read clipboard data only when explicitly triggered by the user, such as when pasting an ICAO code, callsign, route string, global key, frequency, waypoint, or aviation-related identifier into the extension.

Clipboard contents are used temporarily for parsing and lookup features. Clipboard data is not sold, shared, or used for tracking.

Downloads

SandCat may use the browser downloads permission to allow the user to export or download aviation-related data, generated files, logs, or lookup results when the user initiates that action.

Notifications, Alarms, and Timed Tasks

SandCat may use alarms for scheduled extension tasks, periodic local checks, refresh reminders, cached-data updates, or user-enabled aviation workflow features.

3. Information SandCat Does Not Collect

SandCat does not collect:

  • Names
  • Email addresses
  • Physical addresses
  • Payment information
  • Personal communications
  • Personal browsing history
  • Analytics or telemetry data

4. How Information Is Used

Data processed by SandCat is used only to provide extension functionality, including:

  • Airport, runway, waypoint, navaid, procedure, and frequency lookup
  • ADS-B flight tracking overlays
  • SkyVector, ForeFlight, and aviation map overlays
  • Aviation dataset caching and searching
  • Labelbox workflow utilities
  • LiveATC and aviation audio/frequency research tools
  • Locally saved settings and preferences
  • User-directed exports or downloads

5. Permissions Justification

Storage

Used to save preferences, cached aviation datasets, and temporary session state locally.

Tabs and Active Tab

Used to detect and interact with supported aviation-related pages when the user is actively using the extension.

Scripting

Used to inject SandCat scripts into supported pages to provide overlays, extract visible aviation-related information, and enable extension functionality.

Clipboard Read

Used only when the user initiates an action involving pasted aviation-related data.

Alarms

Used for scheduled local extension tasks, refreshes, reminders, cached-data updates, or user-enabled aviation workflow features.

Downloads

Used only when the user initiates a download or export from SandCat.

6. Host Permissions Justification

SandCat requests access to specific domains to provide aviation data, overlays, airport lookup tools, workflow utilities, and user-directed extension features.

Aviation Data Sources

  • davidmegginson.github.io / ourairports.com
    Used to retrieve OurAirports aviation datasets.
  • faa.gov / aeronav.faa.gov / nfdc.faa.gov
    Used to access FAA aeronautical data, NASR data, procedures, and CIFP-related files.
  • airnav.com / opennav.com
    Used for airport lookup and supplemental airport information.
  • airservicesaustralia.com
    Used to access Australian aviation charts, procedures, and waypoint data.
  • raw.githubusercontent.com
    Used to retrieve SandCat-compatible aviation datasets or extension-maintained data files.

Aviation Tools and Overlays

  • skyvector.com
    Used to support SkyVector overlays and aviation-related visualization.
  • globe.adsbexchange.com
    Used to support ADS-B Exchange overlays and selected-aircraft context tools.
  • plan.foreflight.com
    Used to support ForeFlight planning overlays and related aviation workflow features.
  • liveatc.net
    Used to support aviation frequency and audio lookup workflows.

Workflow and Utility Services

  • labelbox.com
    Used only when the user is actively using Labelbox. SandCat may extract visible task or data-row identifiers to assist the user’s workflow.
  • firestore.googleapis.com
    Used only for extension-supported cloud data features, if enabled by the user or required by a SandCat feature.
  • ntfy.sh
    Used only for user-directed notification or alert features.

Access to these domains is used only for SandCat functionality. SandCat does not sell, rent, trade, or share user data.

7. Broad Page Access

SandCat may use broad match patterns such as <all_urls> for certain content scripts and web-accessible resources to detect supported environments and enable overlays or workflow tools.

These scripts perform limited checks to determine whether a page is relevant to SandCat functionality. They are not used to collect, store, or transmit unrelated browsing data.

8. Web Accessible Resources

SandCat includes bundled aviation datasets and local resources such as:

  • Airport datasets
  • Waypoint and navaid files
  • Procedure and frequency datasets
  • Australian, Canadian, Irish, Dutch, and Swiss aviation data
  • Visual waypoint datasets
  • Locally packaged scripts and overlays

These resources are packaged with the extension and are used to provide SandCat functionality. They do not collect personal information.

9. No Remote Code Execution

SandCat does not execute remote code. All scripts and application logic are packaged with the extension.

The extension may retrieve aviation data or user-directed external data from approved sources, but it does not dynamically load or execute external JavaScript.

10. No Analytics or Tracking

SandCat does not use analytics, tracking tools, advertising identifiers, or telemetry.

  • No usage analytics are collected
  • No behavioral tracking is performed
  • No data is sent to analytics services

11. Data Sharing

SandCat does not sell, rent, trade, or share user data.

12. Third-Party Websites

SandCat may interact with third-party aviation websites and workflow tools. These websites operate under their own privacy policies. SandCat only accesses these sites to provide user-requested extension functionality.

13. Changes to This Policy

This privacy policy may be updated from time to time. Updates will be reflected on this page with a revised “Last updated” date.

14. Contact

For questions about this privacy policy, please contact the developer through the GitHub repository associated with SandCat.
