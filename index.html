Privacy Policy for SandCat

Last updated: April 30, 2026

1. Introduction

SandCat is a browser extension designed to assist with aviation data lookup, live aviation overlays, ADS-B analysis, airport information, workflow utilities, and related aviation research tools.

2. Information We Collect

Local Data Storage

SandCat may store data locally on your device using browser storage. This may include:

  • User preferences and settings
  • Cached aviation data, such as airports, runways, waypoints, navaids, and procedures
  • Temporary session data, such as selected aircraft or active flight context

This data remains on your device and is not transmitted to any SandCat-owned server.

Clipboard Access

SandCat may read clipboard data only when explicitly triggered by the user, such as when pasting an ICAO code, callsign, route string, global key, or other aviation-related identifier into the extension.

Clipboard contents are used temporarily for parsing and lookup features. Clipboard data is not stored, transmitted, sold, or shared.

No Personal Data Collection

SandCat does not collect:

  • Names
  • Email addresses
  • Physical addresses
  • Payment information
  • Personal browsing history
  • Personal communications

3. How Information Is Used

Data processed by SandCat is used only to provide extension functionality, including:

  • Aviation lookup features
  • Airport, runway, waypoint, navaid, and procedure search
  • ADS-B flight tracking overlays
  • Map overlays on supported aviation websites
  • Workflow tools such as Labelbox ID extraction
  • Locally saved settings and preferences

4. Permissions Justification

Storage

Used to save preferences, cached aviation datasets, and temporary session state locally.

Tabs, Active Tab, and Web Navigation

Used to detect and interact with supported aviation-related pages when the user is actively using the extension. These permissions allow SandCat to identify relevant tabs, update overlays, and support aviation workflow tools.

Scripting

Used to inject SandCat scripts into supported pages in order to provide overlays, extract visible aviation-related information, and enable extension functionality.

Clipboard Read

Used only when the user initiates an action involving pasted data. Clipboard contents are not stored or transmitted.

5. Host Permissions Justification

SandCat requests access to specific domains in order to provide live aviation data, aviation overlays, airport lookup tools, and user-directed workflow enhancements.

Aviation Data Sources

  • davidmegginson.github.io / ourairports.com
    Used to retrieve OurAirports CSV data, including airport and runway datasets.
  • faa.gov / aeronav.faa.gov / nfdc.faa.gov
    Used to access FAA NASR data, aeronautical data, procedures, and CIFP-related files.
  • airnav.com / opennav.com
    Used for airport lookup and supplemental airport information.

Aviation Tools and Overlays

  • skyvector.com
    Used to support SkyVector map overlays and aviation-related data visualization.
  • globe.adsbexchange.com
    Used to support ADS-B Exchange flight tracking overlays and aircraft context tools.
  • plan.foreflight.com
    Used to support ForeFlight planning overlays and related aviation workflow features.

User-Directed Workflow Tools

  • labelbox.com
    Used only when the user is actively using Labelbox. SandCat may extract visible task or data-row identifiers to assist the user’s workflow. SandCat does not store or transmit Labelbox data externally.

Access to these domains is used only for SandCat functionality. SandCat does not sell, share, or transmit user data to third parties.

6. Broad Page Access

SandCat may use broad match patterns such as <all_urls> for certain content scripts and web-accessible resources to detect supported environments and enable overlays or workflow tools.

These scripts perform limited checks to determine whether the page is relevant. They remain inactive on unrelated websites and do not collect, store, or transmit browsing data.

7. No Remote Code Execution

SandCat does not execute remote code. All scripts and application logic are packaged with the extension and reviewed as part of the Chrome Web Store submission.

The extension does not dynamically load or execute external JavaScript.

8. No Analytics or Tracking

SandCat does not use analytics, tracking tools, or telemetry.

  • No usage analytics are collected
  • No behavioral tracking is performed
  • No data is sent to analytics services

9. Data Sharing

SandCat does not sell, rent, trade, or share user data.

10. Third-Party Websites

SandCat may interact with third-party aviation websites and tools. These websites operate under their own privacy policies. SandCat only accesses these sites to provide the user-requested extension functionality.

11. Changes to This Policy

This privacy policy may be updated from time to time. Updates will be reflected on this page with a revised “Last updated” date.

12. Contact

For questions about this privacy policy, please contact the developer through the GitHub repository associated with SandCat.
